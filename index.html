Document Scanner Project - First Entry

Posted on: October 2025

The idea behind the project is relatively simple, it's to create my own version of "Google Lens", but instead of it being uploaded to the cloud and being processed by massive servers somewhere in the world, I want it to be processed right next to me, in an old mini-PC. It could turn out to be a daring project, yet I feel like I will also learn a lot from this idea, so I'll try to complete this plan.

How will I go about this?

Here is a basic description:

The system runs on a mini-computer with Ubuntu, featuring a Flask web interface for easy access. Here's the technical breakdown:

  • Backend: Python with Flask
  • OCR Engine: Tesseract (open-source)
  • Hardware: Mini-PC + USB camera, probably a "box" either 3D printed or made from wood

Current Progress

During the summer I've developed a proof of concept running on a Linux Subsystem for Windows and using the built-in camera on my laptop. What is left is reprogramming the whole for the mini-PC and the USB-Camera and then streamlining the whole process for the end-user.
